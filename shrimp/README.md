# Shrimp (正式版插件)

该目录是正式版插件源码。当前功能包括：

- 本地坐标驱动的页面聚焦遮罩
- Focused/Scanning/Fatigue 三种自动模式
- 弹窗可调视觉参数与模式阈值
- 校准入口（开始校准 / 重置校准 / 打开后端校准页）

加载方式：

1. 打开 `chrome://extensions`
2. 开启开发者模式
3. 加载已解压扩展，选择本目录
4. 在弹窗中填写后端坐标 URL（例如 `http://127.0.0.1:3000/coordinate`）
